Algorithme de calcul de la suite de Fibonacci

Mathématiquement parlant, la suite de Fibonacci est une suite de nombres entiers dans laquelle chaque terme est la somme des deux termes qui le précedent.
Les termes de cette suite, qui commence par les termes 0 et 1, sont appélés suite de Fibonacci.

Soit la suite F0 = 0, F1 = 1 et Fn = Fn-1 + Fn-2, pour n > 0.
